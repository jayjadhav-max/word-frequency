# Indian Language Word Frequency Analyzer

A comprehensive Python application for analyzing word frequency in Indian languages including Hindi, Tamil, Gujarati, and Marathi.

## 🌟 Features

- **Unicode Support**: Full support for Devanagari, Tamil, Gujarati, and Marathi scripts
- **NLP Techniques**: Tokenization, stopword removal, text normalization
- **Interactive CLI**: User-friendly command-line interface
- **Multiple Input Methods**: Direct text input or file upload
- **Comprehensive Statistics**: Word frequency, lexical diversity metrics
- **Export Formats**: JSON, CSV, and text reports
- **Modular Architecture**: Well-organized, maintainable code

## 🔧 Technologies

- Python 3.6+
- NLTK - Natural Language Toolkit
- NumPy - Numerical computing

## 📋 Supported Languages

1. Hindi (हिन्दी)
2. Tamil (தமிழ்)
3. Gujarati (ગુજરાતી)
4. Marathi (मराठी)
